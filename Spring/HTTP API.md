# HTTP API

## **HTTP API란?**

- HTML이 아니라 데이터를 전달
- 주로 JSON 형식 사용
- 다양한 시스템에서 호출
- 데이터만 주고 받음, UI 화면이 필요하면 클라이언트가 별도로 처리
  - App Client(iPhone, Android, PC App)
  - 웹 브라우저에서 자바스크립트를 통한 HTTP API 호출
  - React, Vue.js 같은 웹 클라이언트
- App, Web Client, Server to Server
- Server to Server
  - 주문 서버 -> 결제 서버
  - 기업간 데이터 통신
